# Postman API Testing

This repository contains sample API tests using Postman collections and Newman for automated testing.

## Structure

- `collections/` - Postman collection files
- `tests/` - Test files with pass/fail scenarios
- `package.json` - NPM dependencies
- `.gitignore` - Git ignore rules

## Setup

```bash
npm install
```

## Run Tests

```bash
npm test
```

## Features

- Sample API endpoint tests
- Pass/Fail assertions
- JSON response validation
- Status code verification
